# 🧮 CalculatorApp

SwiftUI로 제작된 심플한 iOS 계산기 앱입니다.  
UIKit 없이 오직 코드베이스만으로 구성하였으며, 학습 목적 및 연습용으로 적합합니다.

---

## ✨ 주요 기능

- [x] 숫자 입력 (0 ~ 9)
- [x] 사칙연산 지원 (`+`, `-`, `×`, `÷`)
- [x] 초기화 기능 (`AC`)
- [x] 결과 계산 (`=`)
- [x] 연산자 중복 입력 방지
- [x] 실시간 결과 표시 (예: `12 + 3` → `15`)
- [x] SwiftUI `View` 및 `MVVM` 구조 일부 적용
